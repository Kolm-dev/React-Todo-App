
# React Todo App

Приложения для планирования задач

## Команды

Запуск локального сервера для разработки

```bash
  npm install
  npm run dev
```
    
Продакшн версия проекта

```bash
  npm install
  npm run build
```
    

## Технологии

**Front-end:** ReactJS, MUI
